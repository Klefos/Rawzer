# Rawzer

**Raw Data Analyzer for Tecan Infinite®**

A lightweight tool designed to process raw kinetic data exported from **Tecan Infinite** instruments and generate a clean, unified table suitable for downstream analysis.



## **Overview**
Rawzer imports one or more raw files generated by Tecan i-Control and reconstructs a complete kinetic dataset.  
All timepoints belonging to the same well are automatically grouped under a **single, consistent column**, using the **well names defined in the plate layout** rather than generic labels like A1, A2, A3.

This ensures that biological or experimental identifiers (e.g., *Control*, *Treatment A*, *Sample 3*) are preserved throughout the entire workflow.

---

## **Key Features**
- Detection of timepoint structure  
- Alignment of multiple runs from the same experiment  
- Use of **custom well names** instead of A1–H12  
- Merging of all timepoints for each well into a single column  
- Export to XLSX

---

## **Plate Layout Editor**
The application provides an interactive editor for customizing and exporting the microtiter plate layout.

- Each well is displayed as a clickable element.  
- Selecting a well opens a panel where you can assign a **custom name** (e.g., *Control*, *Blank*, *Drug A 10µM*, *Sample 3*).  
- Names are saved as part of the plate layout and applied automatically during data processing.  
- The layout can be reset or modified at any time.

This ensures that the exported dataset reflects the **actual biological or experimental meaning** of each well.

---

## **Data Processing Workflow**
Once the files are loaded, the software performs a structured analysis:

- Parses all OD values for each well  
- Aligns timepoints
- Handles interrupted or restarted kinetic runs  
- Applies the **well names defined in the plate layout**  
- Groups all measurements from each well under a single column  
- Builds a complete, continuous dataset  

This ensures that every well contains the full kinetic profile, labeled with the correct experimental name.

---

## **Output Table Structure**
The final table includes:

- A **Time** column (hours or minutes)  
- One column per well, using **user‑defined names** (e.g., *Blank*, *Control*, *Drug A 10µM*)  
- Each column containing the full kinetic curve of that well  

Example:

| Time (h) | Control | Drug A | Drug B | … |
|----------|----------|-------------|--------------|---|
| 0      | 0.042    | 0.039       | 0.041        | … |
| 1      | 0.050    | 0.048       | 0.049        | … |
| 2      | 0.065    | 0.062       | 0.064        | … |
| …        | …        | …           | …            | … |

This format is ideal for plotting, curve fitting, and comparative analysis.

---

## **Best Practices**
- Always use unmodified raw files exported directly from i-Control  
- Keep all files from the same experiment in a single folder  
- Use consistent naming conventions for experiments and wells  
- Verify the plate layout before processing  
- Export frequently during long kinetic runs to avoid data loss

---

# **Quick Usage Guide**

## **1. Load a Plate Layout**
- Click **Load Layout**.  
- Select the plate layout file you want to use.  
- Assign names to the wells by clicking them in the UI.  
  - If you skip this step, the software will use the **default well coordinates** (A1, A2, …).

---

## **2. Import Raw Data**
- Click **Import Raw Data**.  
- Select the raw file exported from **Tecan i-Control**.  
- The software will automatically parse all wells and timepoints.

---

## **3. Run the Analysis**
- Click **Analyze**.  
- A processed file will be generated automatically with the name:  
  **Rawzer_Processed_UNIXTIME.xlsx**  
  (where *UNIXTIME* is the timestamp of the operation).

---

## **4. Saving the Plate Layout**
- To save your customized plate for future use, click **Save Plate**.  
- The software will generate an Excel file named:  
  **Rawzer_Plate_UNIXTIME.xlsx**  
- This file contains the full layout, including all custom well names.

---

## **5. Resetting the Plate Layout**
- If you want to clear all well names and start from scratch, click **Reset**.  
- All wells will return to their default state.
